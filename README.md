# Inberancar
## Introducción
Inberancar es el nombre de unestro proyecto arduino. El proyecto es un invernadero con ventilación, sensor de humedad, sistema de riego e iluminación.

## Componentes
#### 1.Arduino
#### 2.Motor pequeño de 6V DC
#### 3. 270 Ω Resistencia
#### 4.Transistor PN2222
#### 5.Cables
#### 6.LED (blanco, amarillo)
#### 7.Relé
#### 8.bomba de agua
#### 9.sensor humedad
